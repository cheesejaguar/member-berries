# member-berries

Home directory for the Pi memory project.

## Purpose
This workspace contains the design, templates, extension code, and examples for a local-first project memory system for Pi.

## Structure
- `specs/` — design specs and planning documents
- `templates/` — reusable memory file templates for projects
- `extension/` — Pi extension implementation for memory injection, checkpointing, and search
- `examples/` — example project setups and sample memory files
